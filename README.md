# Bioacoustic Species Detection

Acoustic species identification in the Pantanal, South America. Built for the [BirdCLEF+ 2026](https://www.kaggle.com/competitions/birdclef-2026) Kaggle competition as part of the Master MIND ML program.

The goal is to detect and classify bird and frog vocalizations from passive acoustic monitoring recordings using machine learning.
